Before you begin, make sure you have the following:

- A machine that meets the [system requirements]({{base_path}}/deploy/get-started/install/#system-requirements) (minimum 4 vCPUs, 4 GB RAM, 10 GB disk)
- Java Development Kit (JDK) 11, 17, or 21 installed and set as `JAVA_HOME`
- About 1 hour

!!! tip
    You do not need an external database or load balancer for the evaluation path. WSO2 Identity Server uses an embedded H2 database by default.
