1. Log in to the WSO2 Identity Server Management Console (`https://<IS_HOST>:<PORT>/carbon`) using administrator credentials (`admin:admin`).

2. Navigate to **Main** > **Identity** > **Users and Roles** > **Add**.

3.  Click **Add New User**.
